AuthCognito · Passkeys

Manual test app for PR #708. Run npm run dev for the in-memory mock or npm run sandbox for a real Cognito pool.

Authentication

First time: create an account with username + password, then go to "Manage passkeys" and click "Add a passkey to this device". Returning visit: sign out, then click "Sign in with passkey", enter your username, and approve the prompt.

Tip: Chrome on macOS uses Touch ID / your iCloud Keychain; on Windows it uses Windows Hello. Cross-device QR is also offered automatically by the browser.
Mock-only: sign-up codes appear here (no mailbox).
